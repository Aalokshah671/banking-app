# Insta verify Banking

This project  is a web development project using python(Flask) that simlulate real world banking operations.
The application allows users to securely manage their accounts, perform transactions, and apply for loans through an interactive web interface.

The system includes essential banking features such as user registration, login authentication, deposit, withdrawal, fund transfer, and transaction history tracking. It also integrates a loan application module with a manual admin approval system, making it closer to real industry workflows.

The application is designed with a focus on security, usability, and scalability, ensuring smooth handling of user data and financial operations. Additionally, an admin panel is implemented to monitor users, approve/reject loan requests, and manage system activities.

This project demonstrates practical knowledge of:

Backend development using Flask
Database handling (SQL)
Authentication and session management
Real-world financial system logic
Full project structuring and deployment readiness
